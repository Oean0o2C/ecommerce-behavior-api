# 电商用户行为数据分析平台演示脚本

## 1. 项目概述

### 1.1 项目背景
- 基于Kaggle电商用户行为数据集（eCommerce behavior data from multi category store）
- 构建完整的数据工程链路，实现从原始数据清洗到可视化展示的端到端数据平台
- 对标企业级数据仓库建设要求，覆盖离线数据处理、维度建模、API服务化、前端可视化等核心环节

### 1.2 项目目标
- **功能目标**：构建可在线访问的电商数据分析平台，支持用户行为漏斗分析、实时销售看板、用户画像查询
- **技术目标**：掌握云原生数据架构，实践现代数据工程工具链（Supabase/FastAPI/Streamlit）
- **职业目标**：产出可演示的作品集，支撑大数据研发工程师岗位面试

### 1.3 项目范围
- **包含范围**：千万级原始数据的清洗与转换（ETL）、维度建模与数据仓库构建、RESTful API设计与实现、交互式数据可视化、云基础设施部署与运维
- **不包含范围**：实时流处理（预留扩展接口）、机器学习模型（项目三扩展）、用户权限与支付系统

## 2. 技术架构

### 2.1 总体架构
- **数据处理层**：Kaggle Notebook（分块读取、清洗转换、批量写入）
- **数据存储层**：Supabase PostgreSQL（星型模型、物化视图、索引优化）
- **API服务层**：FastAPI（RESTful接口、异步处理、自动文档）
- **可视化层**：Streamlit（交互式仪表盘、数据可视化、用户友好界面）

### 2.2 数据架构
- **星型模型**：事实表（fact_user_behavior）+ 维度表（dim_time、dim_users、dim_products）
- **物化视图**：mv_daily_sales（日销售汇总）、mv_category_stats（品类统计）、mv_user_rfm（用户RFM分层）
- **数据流向**：Kaggle原始CSV → 分块读取与清洗 → 维度累积统计 → 事实表分块暂存 → 本地输出目录 → 本地导入 → Supabase/PostgreSQL

### 2.3 技术栈
- **后端**：Python 3.11+, FastAPI 0.110+, SQLAlchemy 2.0+, Pydantic 2.0+
- **前端**：Streamlit 1.32+, Plotly 5.18+
- **数据库**：Supabase PostgreSQL
- **部署**：Railway（后端）、Streamlit Cloud（前端）

## 3. 核心功能演示

### 3.1 ETL数据处理
- **数据清洗**：空值处理、异常值过滤、格式标准化
- **数据转换**：时间维度拆解、类目层级解析、价格归一化
- **数据加载**：批量写入云数据库、事务控制、数据质量检查

### 3.2 API接口
- **销售概览**：GMV、订单数、UV、转化率
- **销售趋势**：按日/周/月粒度查询
- **品类分析**：各品类销售额占比
- **用户行为漏斗**：浏览→加购→购买转化率
- **用户RFM分层**：基于最近购买时间、购买频率、购买金额的用户分层
- **热销商品**：基于销售额、浏览量、加购量的商品排序

### 3.3 前端可视化
- **销售仪表盘**：KPI卡片 + 趋势折线图
- **品类分析**：饼图/柱状图/树图
- **用户行为漏斗**：漏斗图 + 转化率标注
- **时间范围选择**：最近7天/30天/自定义
- **数据下钻**：从品类点击看商品明细
- **数据导出**：CSV/Excel下载

## 4. 部署与运维

### 4.1 环境划分
- **Dev**：本地开发（localhost:8000, localhost:8501）
- **Staging**：集成测试（Railway dev, Streamlit Cloud dev）
- **Prod**：生产演示（Railway prod, Streamlit Cloud prod）

### 4.2 CI/CD流程
- **GitHub Push/PR** → **GitHub Actions**（代码格式检查、单元测试、构建Docker镜像） → **Railway部署**（自动部署main分支） → **Streamlit Cloud**（前端自动部署）

### 4.3 成本估算
- **Supabase**：免费额度内（使用抽样子集）
- **Railway**：~$2/月（轻量服务）
- **Streamlit Cloud**：1个公开应用（免费）
- **总计**：$0/月
